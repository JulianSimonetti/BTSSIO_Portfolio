La veille

Au cours de ma formation et à travers différentes situations (notamment ma mission de stage et projets en cours de formation) j'ai été amené à effectuer de la veille technologique. La veille technologique consiste à rester informé·e de l'actualité d'une technologie, méthode ou ensemble de méthodes particulières.
On a recours pour cela à différents outils selon les préférences et les formats utilisées.
Dans mon cas j'ai eu principalement recours au suivi de flux RSS de différents sites d'actualité informatique (exemples : silicon.fr, usine-digitale.fr, developpez.com, zdnet.fr, cnetfrance.fr, ainsi que divers sites anglophones). Ces flux sont regroupés dans un agrégateur de flux, je me suis personnellement servi de Feedly. J'ai associé cette surveillance de flux à de nombreuses et régulières recherches personnelles.

Les technologies de développement web

Le développement web se base sur l'utilisation de langages reconnus par tous les navigateurs : HTML pour la mise en page, CSS pour le style, JavaScript pour le traitement côté client. Par ailleurs du PHP est utilisé côté serveur dans le cas des sites web dynamiques.
En revanche, et comme pour la plupart des langages informatiques, des bibliothèques et frameworks existent et permettent de faciliter et d'approfondir le développement web.

L'évocation du HTML6

Depuis peu, les discussions s'ouvrent concernant un possible passage à l'HTML6, et le W3C a ouvert sa plateforme aux propositions.
Un développeur web a récemment lancé l'idée de modifier certains aspects du HTML avec comme objectif principal d'y apporter les fonctionnalités de l'actuel JavaScript. La finalité serait de se débarrasser du JS ou au minimum de limiter son utilisation au nécessaire, en permettant les traitements les plus courants d'être intégrés au HTML et compris directement à cette étape de l'interprétation par le navigateur.

Le HTML6 : la fin de JavaScript ?

Cela serait une révolution dans un domaine où la popularité de JavaScript est conséquente : il est en 2020 le langage le plus utilisé par les développeurs, juste devant Python et Java.
Cependant cela ne fera qu'animer les discussions autour de la nouvelle version du standard HTML, et si le JavaScript ne disparaîtra peut-être pas de sitôt, il faut s'attendre à de nombreuses évolutions.

Le framework Symfony

Le framework Symfony : un outil bientôt indispensable

Le développement d'un site web dynamique n'est pas une tâche aisée et encore moins rapide. Cependant, avec des méthodes souvent communes, certains frameworks émergent pour aider les développeurs.
Le framework le plus populaire à l'heure actuelle est Symfony. Il permet de développer un site dynamique en PHP basé sur l'architecture MVC, avec différentes fonctionnalités adaptables aux cahiers des charges possibles. Il est utilisé par de très nombreuses plateformes telles que Google, Facebook ou encore le projet Wikimédia relié au site Wikipédia. Il est davantage enseigné dans les universités et sa maîtrise est un atout recherché par les entreprises du monde entier.
